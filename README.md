**使用KVStore_beta**

1. **包含头文件KVStore_beta.h**: include "KVStore_beta.h"
2. **创建KVStore_beta对象**: KVStore_beta store("datapath");
3. **在cmake项目中链接KV_STORAGE_BETA库**:  target_link_libraries(storage_example KV_STORAGE_BETA)
